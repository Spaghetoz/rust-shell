# Rust (mini)-shell

### A basic unix shell made with Rust

### Supported platforms: 
- Linux (tested on Ubuntu WSL)
- Maybe MacOS

### Features
- Commands parsing
- Commands execution : 
    - Simple commands (for example `ls -l /`)
    - Basic redirections (<, >>, >>, 2>)
    - Pipes commands
- Pipes chaining and partially redirections chaining

### How to use : `cargo run`
